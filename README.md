# main-pg-archive

## 项目简介

`main-pg-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定网站或域名，仅作为静态 HTML 页面的集中存储与展示空间。

## 目录结构

```
main-pg-archive/
├── pages/              # 存放所有归档的 HTML 页面
├── assets/             # 公共资源文件（样式、脚本、图片等）
├── index.html          # 仓库主页，可浏览所有归档页面
└── README.md           # 本文件
```

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，位于 `pages/` 目录下。
- 每个页面可能包含独立的样式与脚本，也可引用 `assets/` 下的公共资源。
- 页面文件名应具有描述性，便于识别内容主题。
- 归档页面仅供存档与展示用途，不保证持续更新或维护。

## 维护说明

- 欢迎提交归档页面，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的链接列表。
- 提交前请确保页面内容合法合规，不包含恶意代码或敏感信息。
- 本仓库由维护者不定期审查与整理，可能对页面结构或内容进行清理。

## 使用方式

1. 克隆本仓库到本地。
2. 在浏览器中打开 `index.html` 即可查看所有归档页面列表。
3. 也可直接访问 `pages/` 下的单个 HTML 文件。
